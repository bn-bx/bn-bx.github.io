Windows Will Not Boot After Cloning to New HDD or SSD?

  1. Boot into Windows Recovery Enviroment via USB or CD.

  2. Once there, choose the following options; Repair your Computer > Troubleshoot > Advanced > Command Prompt

  3. Type the following command: diskpart

  4. Type the following command: list disk

  5. Type the following command: sel disk # (replace # with the number of the disk that has Windows on it.)

  6. Type the following command: list vol

  7. Type the following command: sel vol # (replace # with the EFI volume, usually a couple hundred mb in size.)

  8. Type the following command: set id=c12a7328-f81f-11d2-ba4b-00a0c89ec89b

  9. Type the following command: assign letter=A:

  10. Type the following command: exit

  11. Now that you have exited Diskpart type the following command: bcdboot C:\Windows /l en-us /s A: /f ALL (keep a eye on the first part of the command that includes drive letter C, sometimes in the RE the normal C drive can be mounted as D or something else double check this by listing volumes in diskpart otherwise the command may not be successful)

  12. Exit and reboot, you are finished!
